## IPLMS Office Converter 솔루션 소개 슬라이드 (5장)

---

### **슬라이드 1: 타이틀 슬라이드**

**제목:** IPLMS Office Converter : 자동화된 문서 변환 및 데이터 추출 솔루션

**부제:** GUI 및 REST API를 통한 유연한 오피스 문서 처리

**주요 기능:**
*   Java 기반 데몬 및 GUI 하이브리드
*   LibreOffice를 활용한 PDF 변환
*   PDF 텍스트 추출
*   REST API를 통한 실시간 단일 파일 변환
*   안전한 동시성 제어

---

### **슬라이드 2: 솔루션 개요 및 핵심 가치**

**제목:** 문서 처리의 새로운 표준: 효율성과 접근성 극대화

**핵심 문제점:**
*   다양한 오피스 문서(MS Office, 한컴오피스)의 수동 변환 및 관리의 비효율성
*   문서 내 텍스트 데이터 추출 및 활용의 어려움
*   시스템 간 문서 변환 연동의 복잡성

**IPLMS Office Converter 솔루션:**
*   **Java 기반 유틸리티**: 안정적이고 확장 가능한 백엔드 처리.
*   **자동화된 문서 변환**: 지정된 폴더의 문서를 주기적으로 PDF로 변환하고 텍스트를 추출.
*   **하이브리드 제어**: 사용자 친화적인 GUI와 시스템 연동을 위한 REST API 제공.
*   **핵심 가치**:
    *   **효율성 증대**: 수동 작업 제거, 시간 및 비용 절감.
    *   **데이터 접근성 향상**: PDF 및 텍스트 추출을 통한 정보 활용 극대화.
    *   **표준화**: 모든 문서를 PDF 형식으로 통일하여 관리 용이.

---

### **슬라이드 3: 핵심 기술 및 기능**

**제목:** 강력한 엔진과 유연한 인터페이스

**1. Java 기반 데몬 및 GUI 하이브리드:**
*   **데몬 모드**: 백그라운드에서 상시 실행되며, 설정된 주기에 따라 자동 변환 작업 수행.
*   **GUI (ConverterGUI)**:
    *   800x600 크기의 직관적인 사용자 인터페이스.
    *   서비스 시작/종료, 실시간 로그 모니터링 및 초기화 기능.
    *   `JTextPane`을 활용한 로그 태그별 색상화로 가독성 극대화.

**2. LibreOffice 엔진을 통한 고품질 PDF 변환:**
*   **지원 포맷**: `docx`, `doc`, `xlsx`, `xls`, `pptx`, `ppt`, `hwpx`, `hwp` 등 다양한 오피스 문서.
*   **Headless 모드**: 서버 환경에 최적화된 비시각적 변환 처리.
*   **정확성**: 원본 문서의 레이아웃과 서식을 최대한 유지하며 PDF로 변환.

**3. Apache PDFBox를 활용한 PDF 텍스트 추출:**
*   변환된 PDF 파일에서 모든 텍스트 콘텐츠를 정확하게 추출하여 `.txt` 파일로 저장.
*   추출된 텍스트는 검색, 분석, 데이터베이스 저장 등 다양한 후처리 작업에 활용 가능.

**4. 내장 REST API를 통한 실시간 단일 파일 변환:**
*   `com.sun.net.httpserver.HttpServer` 기반의 경량 웹 서버 내장.
*   `/api/convert` 엔드포인트를 통해 외부 시스템에서 파일 경로를 전달받아 즉시 변환 수행.
*   변환 결과(PDF 경로, TXT 경로, 추출 텍스트 내용, 소요 시간)를 JSON 형식으로 응답.
