## 1. 개요 및 요구사항 (Requirements)

### 1.1 개요
HWP 및 HWPX 포맷의 한글 문서를 PDF 형식으로 정밀 변환한다.

### 1.2 요구사항 상세

## 2. 기술 설계 (Technical Design)

### 2.1 설계 요약
- LibreOffice에서 hwp 필터를 정식 지원하는지 버전을 확인하거나, 필요 시 전처리 필터 적용 후 `soffice --convert-to pdf:writer_pdf_Export` 옵션으로 실행.
