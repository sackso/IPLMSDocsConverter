## 1. 개요 및 요구사항 (Requirements)

### 1.1 개요
변환이 완료된 PDF 파일의 텍스트 레이어를 파싱하여 동일한 파일명의 .txt 파일로 저장하는 후처리 기능을 수행한다.

### 1.2 요구사항 상세
- [ ] PDF 라이브러리를 통한 텍스트 레이어 데이터 추출
- [ ] 추출된 텍스트 데이터를 UTF-8 인코딩의 .txt 파일로 생성
- [ ] OCR 연동 가능성을 고려한 텍스트 가독성 확보

---

## 2. 기술 설계 (Technical Design)

### 2.1 설계 요약
- Apache PDFBox 라이브러리의 `PDFTextStripper` 클래스를 활용하여 텍스트를 파싱한 후, FileWriter를 사용해 지정된 Output 디렉토리에 저장.

### 2.2 예정 사항 (Next Steps)
- [ ] AutoCAD(DWG/DXF) 파일을 PDF로 변환하는 모듈 연구 및 추가 설계 (LibreOffice 외에 제3의 렌더러 연동 검토)
- [ ] 서버 구동 환경(OS별 LibreOffice 패키지 설치 및 폰트 수동 설치)을 정리한 설치 가이드(Setup Guide) 문서 작성
